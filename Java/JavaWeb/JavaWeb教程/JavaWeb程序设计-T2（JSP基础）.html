一、JSP概述

1、JSP概念

JSP（Java Server Page）是sun公司倡导建立的一种动态网页标准。

用于开发动态网页（将后端开发语言嵌入带前端中【将java嵌入到HTML中】）

2、JSP工作原理

JSP就是将传统Java代码嵌入到html页面代码中，由Web服务器进行编译执行，生成最终的静态html返回客户端。

浏览器无法直接运行JSP文件，只有将包含JSP文件的Web项目部署到Web服务器上，才能看到JSP的显示效果。

3、创建JSP文件

二、JSP脚本标识

JSP脚本标识主要用于将Java代码嵌入到HTML中，主要包括：

  • 声明标识
  • Scriptlet脚本程序
  • JSP表达式

1、声明标识

声明标识用于在 JSP 页面中定义全局的变量或方法，通常使用该标识定义整个 JSP 页面需要引用的变量或方法。

xxxxxxxxxx
<%! 声明变量或方法代码 %>

2、Scriptlet脚本程序

Scriptlet（代码片段）脚本程序，是在 JSP 页面中嵌入的Java代码或脚本代码。

代码片段将在页面请求的处理期间被执行，通过 Java 代码可以定义变量或流程控制语句等。

xxxxxxxxxx
<% 表达式或者变量名 %>

3、JSP表达式

JSP 表达式用于向页面中输出信息，类似于System.out.println()括号中的内容。

xxxxxxxxxx
<%= 表达式或者变量名 %>

【案例】

xxxxxxxxxx
<%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title></title>
</head>
<body>
    <%!
        // 定义方法
        public static void test(){
            System.out.println("run test()...");
        }
    %>
    
    <%
        // 调用方法
        test();
        // 定义变量
        String name = "张三";
        int age = 20;
    %>
    
    <!-- 获取变量值，并嵌入到标签中 -->
    <h1><%=name %></h1>
    <h1><%=age %></h1>
</body>
</html>

三、JSP指令标识

指令标识主要用于设定在整个 JSP 页面范围内都有效的相关设置信息（页面格式的限制），它是被服务器解释并执行的，不会产生任何内容输出到网页中。

基本语法：

xxxxxxxxxx
<%@指令名 属性1="属性值1" 属性2="属性值2"...%>

1、page指令

1.1 设置JSP页面的语言及字符集编码

xxxxxxxxxx
<%@ page language="java" contentType="text/html; charset=utf-8" pageEncoding="utf-8"%>

参数说明：

  • language ：JSP页面使用的后端语言，目前只支持Java
  • pageEncoding ：JSP页面的编码格式
  • contentType ：设置JSP页面的MIME类型和字符编码（MIME：媒体类型，是一种标准，用来表示文档、文件或字节流的性质和格式。）

1.2 在JSP页面导入Java资源

xxxxxxxxxx
<%@page import="包路径"%>

2、include指令

include 指令可以在一个 JSP 页面中包含另一个页面，常用于多页面共同的部分抽离出来（如顶部导航、底部版权等），使用时直接引用，以提升开发维护效率。

xxxxxxxxxx
<%@include file="包含文件路径"%>

3、taglib指令

在JSP文件中，可以通过 taglib 指令声明该页面中所使用的标签库，同时引用标签库，并指定标签前缀，在页面中引用标签库后，就可以通过前缀引用标签库中的标签。

xxxxxxxxxx
<%@taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>

四、JSP注释

JSP文件中会出现三种注释：JSP标准注释、Java注释、HTML注释

1、JSP标准注释

xxxxxxxxxx
<%-- jsp标准注释 --%>

2、Java注释

xxxxxxxxxx
// 单行注释
/* 多行注释 */

3、HTML注释

xxxxxxxxxx
