JavaWeb程序设计-T4（JavaBean和JSP标准动作）

一、JavaBean

1、概念

JavaBean就是一个特殊的Java类【需要遵循Java相关编写规范】，在实际开发中，JavaBean主要用于定义实体类（类似于封装类），主要用于传递数据（将数据放在实体类中统一管理），也会用来处理业务逻辑。

2、JavaBean 创建流程【编写的规范】

创建一个JavaBean，通常需要实现以下要求：

（1）实现序列化接口（Serializable）；

（2）私有化实体类属性；

（3）提供公共的set和get方法供外部访问；

（4）创建无参、全参构造方法

3、JSP 搭配 JavaBean 的优点

（1）将HTML和Java分开，方便后期维护；

（2）将经常用的程序写成JavaBean组件，使用时直接调用，增加程序的复用性，节省开发时间。

二、JSP标准动作

在jsp中使用JavaBean有两种方式：

  • 使用Scriptlet（JSP脚本程序）实现；
  • 使用JSP标准动作实现。

【前导案例】

  • 在Student中创建一个学习的方法
xxxxxxxxxx
public String study() {
    return "学生正在认真学习";
}

  • 方式1：使用 <% %> 调用 JavaBean 中的 study() 方法

    xxxxxxxxxx
    <%
        Student stu = new Student();
        String msg = stu.study();
        out.write(msg);
    %>

  • 方式2：使用 JSP标准动作 调用 JavaBean 中的 study() 方法

    xxxxxxxxxx
    <jsp:useBean id="stu" class="com.model.Student"
                     scope="application">
    </jsp:useBean>
    

    <h1><%=stu.study() %></h1>

1、概述

为方便使用，JSP定义了一系列的标准动作来操作JSP文件，每个标准动作实现不同的功能。JSP标准动作使用为前缀，以标签的形式，可以与HTML标签融为一体。

常用的JSP标准动作如下：

2、JSP 常用标准动作

2.1 <jsp:useBean>

在JSP页面上创建一个Bean实例（创建对象）。

参数说明：

  • id：实例名（对象名）

    class：路径（包名+类名）

    scope：Bean实例的存储范围

    • page：当前页面；
    • request：发生请求时；
    • session：同一会话；
    • application：整个服务器范围内
xxxxxxxxxx
//创建Student的实例（对象）
<jsp:useBean id="stu" class="com.model.Student"
                 scope="application">
</jsp:useBean>

2.2 <jsp:setProperty>

用于设置中指定 Bean 的属性值。 相当于：java 中的 set() 方法

参数说明：

  • name：useBean中的id值（对象名）
  • property：属性名称
  • param：请求的参数名【JSP表单标签的name属性】

2.3 <jsp:getProperty>

用于获取中指定 Bean 的属性值。相当于：java 中的 get() 方法

参数说明：

  • name：useBean中的id值（对象名）
  • param：中设置的property值

【案例】

test.jsp

xxxxxxxxxx
<%@ page language="java" contentType="text/html; charset=UTF-8"
 pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title></title>
    </head>
    <body>
        <form action="show.jsp" method="post">
            姓名：
            <input type="text" name="sname"/><br /><br />
            <input type="submit" value="获取数据"/>
        </form>

show.jsp

xxxxxxxxxx
<%@ page language="java" contentType="text/html; charset=UTF-8"
 pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title></title>
    </head>
    <body>
        <!-- 创建Student实例 -->
        <jsp:useBean id="stu" class="com.model.Student"
                 scope="application">
        </jsp:useBean>


        <!-- 获取属性并设置属性值 -->
        <jsp:setProperty name="stu" property="name" param="sname"/>


        <!-- 获取属性值 -->
        <jsp:getProperty name="stu" property="name"/>
    </body>
</html>

2.4 <jsp:forward>

用于将用户的请求转发到一个HTML文件、JSP文件或Servlet中。

xxxxxxxxxx
<%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title></title>
    </head>
    <body>
        <!-- 将用户请求转发到login.jsp中 -->
        <jsp:forward page="login.jsp"></jsp:forward>
    </body>
</html>

2.5 <jsp:include>

在jsp文件中包含其他文件。

参数说明：

  • page：资源路径
  • flush：是否刷新，值必须为：true
xxxxxxxxxx
<%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title></title>
    </head>
    <body>
        <h1>获取用户数据</h1>
        <!-- 将login.jsp包含值当前页面中 -->
        <jsp:include page="login.jsp"></jsp:include>
    </body>
</html>
