一、JSTL概述

JSTL（Java server pages Standarded Tag Library ），jsp标准标签库，主要用于简化和替换jsp页面上的java代码。

JSTL使用步骤：

（1）导入jstl相关jar包

下载地址：

  • jstl-1.2.jar ：jstl

    • standard-1.1.2.jar ：standard

（2）引入标签库 ：<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>

（3）使用标签

二、JSTL核心标签库

1、c:if 标签【重点掌握】

c:if 标签相当于Java中的 if ，基本语法为：

xxxxxxxxxx
<c:if test="条件">
 ......
</c:if>

test为必须属性，接受boolean表达式，如果表达式为true，则显示if标签体内容，如果表达式为false，则不显示if标签体内容。

c:if 标签没有else情况，如果想要else，可以再定义一个c:if 标签

xxxxxxxxxx
<%
    request.setAttribute("num",10);
%>


<c:if test="${num%2==0}">
    ${num}是偶数
</c:if>

2、c:choose 标签

c:if 标签相当于Java中的 switch ，基本语法为：

xxxxxxxxxx
<c:choose>
 <c:when test="条件">满足条件执行的操作</c:when>
 ...
 <c:otherwise>不满足所有取值是执行的操作</c:otherwise>
</c:choose>

  • c:choose ---> 相当于java中的switch

  • c:when ---> 相当于java中的case

  • c:otherwise ---> 相当于java中的default

xxxxxxxxxx
<%
request.setAttribute("num",7);
%>
<c:choose>
    <c:when test="${num == 1}">星期一</c:when>
    <c:when test="${num == 2}">星期二</c:when>
    <c:when test="${num == 3}">星期三</c:when>
    <c:when test="${num == 4}">星期四</c:when>
    <c:when test="${num == 5}">星期五</c:when>
    <c:when test="${num == 6}">星期六</c:when>
    <c:when test="${num == 7}">星期天</c:when>
    <c:otherwise>输入有误</c:otherwise>
</c:choose>

3、c:forEach 标签【重点掌握】

c:forEach标签相当于Java中的 for循环，主要有两个用途：

  • 执行重复操作，属性包括：

    • begin ： 开始值（包含）

    • end ：结束值（包含）

    • var ： 临时变量

    • step ：步长

xxxxxxxxxx
<%--  输出1-100 --%>
<c:forEach begin="1" end="100" var="i" step="1">
    ${i}<br>
</c:forEach>

  • 遍历容器，属性包括：

    • items ： 容器对象

    • var ： 容器元素的临时变量

    • varStatus ：循环状态对象

      • index ：容器中元素的索引，从0开始

      • count ：循环次数，从1开始

xxxxxxxxxx
<%
    List list = new ArrayList();
    list.add("1001");
    list.add("张三");
    list.add("男");
    list.add(23);
    request.setAttribute("list",list);
%>


<c:forEach items="${list}" var="li" varStatus="s">
    索引：${s.index} 循环次数：${s.count} 容器值：${li}<br>
</c:forEach>

4、c:out 标签

c:out 标签用于显示一个表达式的结果，和<%= %>类似

参数说明：

  • value：要输出的表达式

  • escapeXml：是否将结果中的字符串转换成实体代码

5、c:set 标签

c:set 标签用于设置页面中的变量或对象属性值

参数说明：

  • var：存储变量值的变量名称

  • value：要存储的变量值

6、c:remove 标签

c:remove 标签用于删除创建的变量

参数说明：

  • var：要删除的变量名称

三、JSTL格式化标签库

使用格式化标签库，需要引入fmt标签库：

xxxxxxxxxx
<%@taglib prefix="fmt" uri="http://java.sun.com/jsp/jstl/fmt" %>

1、fmt:formatNumber 标签

fmt:formatNumber 标签用于格式化数字

参数说明：

  • value：指定被格式化的数字

  • type：数字格式的表现类型

    • number：数字格式

    • currency：货币格式

    • percent：百分比格式

  • maxFractionDigits：保留小数点位数

2、fmt:formatDate 标签

fmt:formatDate 标签用于格式化日期

参数说明：

  • value：指定被格式化的日期

  • pattern：格式化样式
