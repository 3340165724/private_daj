# Series 和 DataFrame 的区别

------------

<br>

- Series是一维的，相当于数组
- DataFrame是二维的，相当于表
- [DataFrame ](../3-DataFrame/dataframe.md "DataFrame") 由 [Series ](../2-Series/series.md "Series") 对象组成的字典
